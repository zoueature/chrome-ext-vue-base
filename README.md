# chrom-ext-base

简单的vue开发chrome检查的相关框架和api封装
```shell
    git clone https://github.com/zoueature/chrome-ext-vue-base.git projectName
    cd productionName
```
## 目录结构
* _locales 多语言支持目录
* api fetch的简单封装
* assets 静态数据
* background background.js脚本
* chrome chrome相关api的封装
* content content.js脚本
* plugins 插件manifest
* popup popup页面
* utils 工具包


## 安装依赖包
```
npm install
```

### Compiles and watch
```
npm run watch
```
